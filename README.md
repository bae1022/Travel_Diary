# Travel_Diary
여행 기록 및 추천 앱

* ## 개요 ##
   * ### 개발 목적 ###
     * 소중한 기억들을 회상하고, 여행을 가본 지역과 가보지 않은 지역을 한 눈에 보게 하고자 개발
   
   * ### 핵심 기능 ###
     1. 여행이나 공연 등 추억들을 기록
     2. 여행했던 지역들을 한 눈에 볼 수 있도록 함
     3. 현재 위치 또는 특정 지역의 여행지를 추천받을 수 있도록 함
     4. 기록했던 사진을 SNS(인스타그램)에 공유할 수 있도록 함
